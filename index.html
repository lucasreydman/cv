MLB BvP

Personal Project

Apr 2026

Sports analytics tool for researching the FanDuel Player Hits (1+) prop (also valid for Total Bases (1+) and Hits (1+) on Bet365 and theScore Bet) by surfacing historical career batter vs. pitcher matchups ranked by batting average weighted for sample size. For any given date, the app loads the MLB schedule and probable starters, builds each lineup using official batting order data from the boxscore when available or falling back to the top nine active players by career plate appearances, and fetches career BvP splits for every batter facing the opposing starter (minimum 10 AB). From each split it computes AVG, OPS, OBP, SLG, and extra-base hits. The UI divides games into Upcoming and In Progress after first pitch, with live refresh keeping the buckets current. A configurable filter panel (Min AB, Min AVG, optional OPS gate) narrows the table to statistically meaningful matchups, and a Top 5 Plays card ranks by AVG × min(AB/30, 1) to balance hit rate with sample reliability. A data quality pass strips team-level aggregate lines that the MLB Stats API occasionally returns in place of true individual BvP, dropping any raw stat line shared by five or more batters on the same team against the same pitcher. The full table is exportable as CSV. Fully responsive: mobile layout swaps the scrollable table for a vertical card list with sort chips, and all game times render in the viewer's local browser time zone. Built with Next.js App Router, React 19, and Tailwind CSS v4. Deployed on Vercel.

Next.js React Tailwind CSS TypeScript MLB Stats API Vercel

Fantasy Draft Lottery Simulator

Personal Project

Mar 2026

Fully configurable fantasy sports draft lottery simulator built with pure vanilla HTML, CSS, and JavaScript, with no frameworks or build step, running entirely in the browser. A six-step setup wizard guides commissioners through league configuration: name, team count, team names, lottery structure, combination weights, and draft rounds. The lottery engine uses recursive state enumeration to compute exact analytical odds for every team at every pick position, handling any number of drawn picks and by-record assignments. Commissioners assign any number of lottery combinations to each team, and the odds table updates instantly to reflect the precise probabilities. Draft pick trades are tracked through a full pick ownership system, so every pick in every round can be reassigned to any team before the lottery runs. The reveal experience is a full-screen animated sequence: quick iterations burn through run 1 through N-1, then the final official result unfolds with a countdown timer, automatic pick reveals from the bottom up, a dynamic podium for the top lottery picks, and upset alerts when a team jumps or falls out of their expected seed range. Results and the complete multi-round draft order can be downloaded as text files. All data persists to localStorage, and league configurations can be reconfigured at any time via the setup wizard. Deployed to GitHub Pages.
