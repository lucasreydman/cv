Pride STEM Canada Website

CSCI 3691 — Intermediate Project · Senior Developer

Winter 2026

Full-stack conference website for Pride STEM Canada, a national nonprofit dedicated to celebrating and advancing the contributions of the 2SLGBTQ+ community in STEM. The organization hosts an annual nationwide conference — now in its sixth year — bringing together 250+ delegates from across Canada for keynote speakers, research presentations, networking events, and career development sessions. As senior developer on a team course project, I led core architectural decisions and owned implementation of the site's most complex features. Built on Next.js with the App Router for full-stack server and client rendering, with React and React Bootstrap powering the component-driven UI and Framer Motion handling smooth page and section animations. Conference registration is fully integrated with Stripe, supporting multiple delegate tiers with secure payment processing. Firebase handles user authentication and real-time data storage, enabling dynamic content management across the site. Resend powers the transactional email pipeline — sending confirmation and notification emails to registered delegates. CSS Modules enforce scoped, collision-free styling across components. The entire codebase is version controlled and deployed through a GitLab CI/CD pipeline. The site serves as the public-facing home for the conference, covering registration, the event schedule, presentation submissions, sponsorship and partnership opportunities, governance documentation, and organizational information — all designed to reflect the mission of building a more inclusive and representative STEM community across Canada.

Next.js React Framer Motion React Bootstrap CSS Modules Firebase Stripe Resend GitLab

QuickCash

CSCI 3130 — Software Engineering · Developer

Sep 2025 – Dec 2025

Full-stack Android mobile application connecting students and part-time workers with short-term job opportunities. Built across two Agile iterations with a team of 8 using strict test-driven development, pair programming, and continuous integration through GitLab. Features role-based access for two user types — employees and employers — each with a tailored dashboard and feature set. Employees can search and filter jobs by salary range, job type, and location (with Google Maps integration), apply with a custom cover letter, track their submission statuses, and receive push notifications for nearby opportunities. Employers can create detailed job postings with requirements, location, employment type, and salary range, review incoming applicants, and accept or decline applications. Both dashboards surface performance insights and application statistics. Also integrates PayPal for payment processing. Firebase powers authentication, real-time data, and Firestore CRUD operations throughout. Code quality enforced through SonarQube refactoring across both iterations, reducing issues from 119 to zero. Architecture follows SOLID principles and applies Singleton, Facade, and Observer design patterns to keep the UI layer cleanly separated from data and auth logic.

Tic Tec Toe – Connecting the Dots in Tech

CSCI 2691 — Introductory Project · Junior Developer

Jan 2025 – Apr 2025

Cross-platform mobile application built for the academic research community, addressing the core challenges researchers face today — information overload from the sheer volume of publications, time constraints that make reading every relevant paper impossible, fragmented discovery across scattered databases and repositories, and a lack of meaningful tools for peer collaboration on existing platforms. Built as a team introductory project over a full semester using React Native and Expo Go, with Supabase providing the backend database, Node.js handling server-side logic, and GitLab managing version control and CI/CD. The platform provides researchers with a centralized, personalized feed of relevant papers tailored to their fields and interests, allowing them to interact with content and peers through likes, comments, and shares. A standout feature is audio playback of research papers, enabling on-the-go consumption for researchers juggling experiments, grant writing, and teaching. The interface was designed iteratively in Figma, with documentation and project proposals written in LaTeX. Project coordination tracked through Jira across the full development lifecycle. The platform's overarching goal is to foster meaningful academic collaboration, accelerate knowledge dissemination across disciplines, and become the go-to space for researchers to stay current, engage with peers, and make significant contributions to their fields.

Personal Project

Vanilla web app that answers the question every student asks at the end of term — what do I need on my final?
